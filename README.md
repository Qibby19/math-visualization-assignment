# EXPLANATION 
- How does visualization help us understand mathematical functions and data?
  
  With visualization, we can quickly analyze the data that we use. As example if the graph shows temperature data, we can quickly analyze the data and know what is the highest, lowest and even mean of the temperature data.

- Which plot was most useful in this assignment and why?
  
  In my opinion all graph has it pros and cons. But for this assignment, scatter graph might be the most useful because with student scores data, we can clearly see the scores and find who is the highest, lowest and in the middle.

- What is the role of NumPy and Matplotlib in your project?
  
  NumPy: Used for efficient numerical operations, especially for creating and manipulating arrays. It's crucial for calculations like total = 0.4 * midterm + 0.6 * final, generating sequences with np.linspace, and performing polynomial fitting with np.polyfit.

  Matplotlib: used for creating various visualizations and plots. This includes generating scatter plots, line plots, histograms, and bar charts to represent the data distributions, relationships between scores, mathematical functions, and predictions. It handles all aspects of plotting, from defining the figure and axes to adding titles, labels, legends, and saving the plots to files.
